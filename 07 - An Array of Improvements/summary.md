# Summary

## Notes

1. `.from()`

    It takes a second argument, which is a map fn.
    
    ```
    const arr = Array.from([1, 2, 3], x => x + x);
    ```
    
2. `.find()`

    `.find()` finds and returns the 1st element. Use `.filter()` to find multiple.
